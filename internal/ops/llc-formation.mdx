## Important notes

**Manager is listed as non-member.** This preserves single-member LLC tax treatment via Schedule C. Do not change this designation without consulting a tax professional.

**Clarely LLC has a separate tax identity from Scrub Hub Crew.** Both file as Schedule C. Consult an accountant before filing. Do not combine them on the same Schedule C without professional guidance.

**Annual report requirement:** Florida single-member LLCs must file an Annual Report with Sunbiz.org each year between January 1 and May 1. Filing fee approximately \$138.75. Failure to file results in administrative dissolution. Add calendar reminder each January.
