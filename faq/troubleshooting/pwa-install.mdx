## I don't see Add to Home Screen on iPhone

The Add to Home Screen option only appears in Safari. If you're using Chrome, Firefox, or another browser on iPhone, it won't appear. Open [clarely.app](https://clarely.app) in Safari and try again.

Make sure you're signed in before trying to install. The install option may not appear on the login page.

## I don't see Add to Home Screen on Android

On Android, use Chrome. Open [clarely.app](https://clarely.app) in Chrome, tap the three-dot menu, and look for **Add to Home screen** or **Install app**. If you don't see it, make sure you're signed in first.

If it still doesn't appear, try clearing Chrome's cache and reloading the page.

## The app installed but opens in a browser tab

If Clarely opens in a browser tab instead of full-screen after installing, the PWA manifest may not have loaded correctly. Try:

1. Uninstall the icon from your home screen
2. Clear your browser cache
3. Open [clarely.app](https://clarely.app) again and reinstall

## The icon looks wrong or shows a generic icon

This happens occasionally when the browser caches an older version. Uninstall and reinstall following the steps above.

## I'm not getting notifications

Push notifications require the app to be installed on your home screen. If Clarely is installed but you're not getting notifications:

- On iPhone: go to **Settings** → **Clarely** → **Notifications** and make sure notifications are enabled
- On Android: go to **Settings** → **Apps** → **Clarely** → **Notifications** and enable
